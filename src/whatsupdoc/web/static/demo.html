Quick Installation

Get started in under 30 seconds with these simple steps.

1. Basic Installation

Add this single line to your HTML page:

<script src="http://localhost:8000/static/widget/whatsupdoc-widget.js"></script> <div id="whatsupdoc-widget"></div>

2. Custom Configuration

Customize the widget with data attributes:

<script src="http://localhost:8000/static/widget/whatsupdoc-widget.js"></script> <div id="whatsupdoc-widget" data-api-url="https://your-api.com" data-theme="dark" data-position="bottom-left" data-title="Ask Our AI" data-primary-color="#10B981"> </div>

Configuration Options

Customize the widget to match your brand and requirements.

Attribute Default Options Description
data-api-url current origin Any valid URL The API endpoint for your WhatsUpDoc instance
data-theme light light, dark, auto Color theme for the widget
data-position bottom-right bottom-right, bottom-left Position of the floating button
data-title Ask WhatsUpDoc Any string Title shown in the chat header
data-placeholder Ask me anything... Any string Placeholder text for the input field
data-primary-color #3B82F6 Any hex color Primary brand color for buttons and accents
